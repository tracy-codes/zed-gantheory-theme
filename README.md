# Gantheory Theme for Zed

A dark theme for [Zed](https://zed.dev), ported from the [VSCode Gantheory theme](https://github.com/shunyaoshih/vscode-theme-gantheory) by shunyaoshih.

## Screenshot

![Gantheory Dark Theme](https://raw.githubusercontent.com/shunyaoshih/vscode-theme-gantheory/master/images/screenshot.png)
*(Screenshot from original VSCode theme)*

## Features

- Very dark background (#040404) for reduced eye strain
- Vibrant syntax highlighting with carefully chosen colors:
  - Cyan keywords (#00afff)
  - Bright cyan functions (#5fffd7)
  - Orange strings (#ffaf5f)
  - Purple numbers (#af00ff)
  - Pink punctuation/operators (#d7005f)
  - Yellow types (#ffd75f)
- Full terminal ANSI color support
- Git status colors

## Installation

1. Open Zed
2. Open the Extensions panel (`cmd+shift+x` or `zed: extensions`)
3. Search for "Gantheory"
4. Click Install
